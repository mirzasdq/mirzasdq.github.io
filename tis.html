Topics in Technology & Information System

Learning Technology and Information Systems in computer science is essential because they provide the foundational knowledge and skills required to design, develop, and manage modern computing systems and solutions.

  • Hardware

    • Knowing how hardware components (like the CPU, memory, storage, and input/output devices) work helps to understand the architecture of computer systems. This foundational knowledge allows me to write more efficient software and understand its limitations.

  • Software

    • From this chapter, I get to learn about the difference of system software, application software, programming software, and malicious software. Learning how to write software teaches critical problem-solving skills. Programming is all about breaking down complex tasks into smaller, manageable pieces, and then finding logical, efficient solutions. These skills are transferable to various other areas of life and work.

  • Information System

    • I get to learn that information system requires people, procedures, software, hardware, data, and network to function. Other than that, this chapter elaborates the functional view of an organisation. For instance, it explains about the management levels and the informational needs for each level in an organisation.

  • Databases and Data Analytics

    • This chapter highlights different types of databases, data analytics techniques, data warehousing, data visualization, and big data analytics. Databases and data analytics are essential for modern businesses to make informed decisions. Data warehousing, visualization, and big data analytics are important components of data analytics. The ability to effectively manage and analyze data is critical for success in today's world of information overload.

  • Network & Communications

    • Networking and communication are critical when designing distributed systems where multiple computers or devices work together. Understanding how these systems communicate helps to design efficient, reliable, and scalable applications.

  • Privacy Security Ethics

    • This chapter covers issues related to the impact of technology on people and how to protect ourselves on the Web. This chapter also covers standards of computer ethics such as copyright and software piracy.

  • Cloud Computing

    • In this chapter, I was able to learn about the basic concepts of cloud computing and the basic cloud service models. There are three main cloud service models which is infrastructure as a service (IaaS), platform as a service (PaaS), and software as a service (SaaS).

  • Reflection

    • I was able to grasp how information systems work. It focuses on the collection, storage, and processing of data. Learning about these systems helps students understand how data flows through an organization or system and how it can be used effectively for decision-making and analysis. This course also focuses on developing students communication by making students approach the industry.
