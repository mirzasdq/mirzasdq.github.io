Topics in Digital Logic

Learning digital logic in computer science is essential because it is the basis of computer architecture. It forms the foundation for understanding how computers and digital systems work at the hardware level.

  • Digital Logic Overview

    • This topic introduced me to logic operations and basic logic functions. At the end of the topic, I learned about some fixed-function integrated circuit and programmable logic devices.

  • Number Systems and Codes

    • Number systems focuses on changing the base of the numbers. It is essential because machine uses binary as their language. This topic also give students in-depth information about code conversion.

  • Logic Gates

    • This chapter provides students detailed steps about how logic gates work. By the end of this topic, students were able to identify the difference between each logic gates.

  • Boolean Algebra & Logic Simplification

    • This chapter focuses on Boolean Expressions and Operations. From this chapter, I learned that there are two ways to simplify an expression. The first one is using Rules of Boolean Algebra and the second one is by using Karnaugh Map.

  • Combinational Logic Circuit

    • This chapter illustrate the conversion between different representation of digital circuit. Other than that, I learned about the universal property of NAND and NOR Gates. It is important to be implemented so that the final circuit use only a single type of gate which will reduce the costs needed to create the circuit. By the end of this chapter, I was able to design a combinational circuits from the problem statements given.

  • Functions of Combinational Logic

    • This chapter elaborates about detailed functions of every combinational logic. Other than that, this chapter also introduced error detection techniques like parity bits which are essential for ensuring data integrity during transmission and storage.

  • Latches, Flip Flop & Timers

    • This chapter starts with the difference between combinational logic circuit and sequential logic circuit. For sequential logic circuit, the storage device is capable of storing binary information unlike combinational logic circuit.

  • Counters

    • In this chapter, I learned about counters which are essential for keeping track of time and events in digital systems. Counters are used in application where it's necessary to count occurrences of events.

  • Shift Registers

    • From this chapter, I learned that shift registers are used to store and transfer data in a sequential manner. They can hold a series of bits and shift them left or right, which is fundamental for data manipulation, storage, and movement within digital systems.

  • Deeds Software

    • This software give students more insight of how the circuit actually works.

  • Reflection

    • I get to learn that computers use binary number system to process and store data. It also teaches how the fundamental operations are carried out at the hardware level. This helps when working with more advanced topic such as memory, data processing, and architecture. Plus, this subject covers about the logic gates in processor.
